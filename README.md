# 👩🏽‍💻 Anita Mocheche — System Administration Portfolio

## 🚀 About Me
I’m a hands-on System Administrator with a strong interest in building, securing, and managing real-world IT environments.

My journey into system administration started with curiosity — and quickly turned into building full lab environments where I simulate enterprise setups, break things (intentionally), fix them, and document everything along the way.

I don’t just configure systems — I focus on **why they are built a certain way**, how they can fail, and how to secure them properly.

## 🔐 My Approach to System Administration
I approach system administration with a strong focus on:

- **Security first** → least privilege, hardening, controlled access  
- **Structure** → clean organization of users, groups, and resources  
- **Automation** → reducing manual work using PowerShell  
- **Troubleshooting** → understanding root causes, not just quick fixes
